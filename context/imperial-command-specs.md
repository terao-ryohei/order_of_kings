# 王の勅命 ゲーム仕様書

## 1. 部隊構成

- **1部隊の構成**: 主将(1) + 副将(1) + 軍師(1)
- **主将サブスキル**: スキルLv7以上でサブスキル2が解放（主将サブスキル1・2、副将サブスキル1・2の計4スロット）
- **軍師の扱い**: 部隊ステータスには直接影響しない。軍師スキルのみを部隊に付与（全体バフ型・特定将領強化型など）
- **コスト制約**: 部隊コスト増加は研究「軍規統制」（国学院Lv4以上）で上昇。最大5部隊まで解放可能（施設進行に応じて順次解放）

### 部隊数別運用方針

| 部隊数 | 推奨構成 |
|--------|---------|
| 1部隊 | 耐久型 or 連鎖武力型 |
| 2〜3部隊 | 耐久部隊 + 火力部隊 |
| 4〜5部隊 | 耐久 + 武力複数型 + 知力型1部隊 |

## 2. 5編成タイプ

| タイプ | 特徴 | 代表主将 | 推奨スキル | 軍師 |
|--------|------|---------|-----------|------|
| **連鎖武力** | 通常攻撃トリガーの連鎖を高速回転させてDPS最大化。2トップ型が強い | 趙雲、岳飛 | 龍胆、閃撃、槍で三関を鎮める | 王彦章（連撃付与）、樊噌（連鎖発動率上昇） |
| **怒気武力** | 時間経過で怒気ゲージが溜まり高倍率スキルを発動。瞬間火力型 | 怒気スキル持ち武将 | 怒気系スキル | 怒気回転強化系 |
| **能動武力** | 発動率依存の爆発力。ターンごとの爆発的ダメージ | 物理攻撃能動系武将 | 能動スキル | 能動発動率強化系 |
| **耐久型** | 被ダメ軽減・回復重視。前衛盾役。1部隊運用時に最推奨 | 胆力・回復系武将 | 被ダメ軽減、回復スキル | 回復・バフ解除系 |
| **知力型** | 知力参照ダメージ。武力型と防御計算が異なるため対応力向上 | 知力型武将 | 知略攻撃スキル | 知略強化系 |

## 3. 兵種相性

### 三すくみ（近接3兵種）

```
刀兵 → 騎兵 → 槍兵 → 刀兵
（→は「強い」方向）
```

| 兵種 | 強い相手 | 弱い相手 | 特徴 |
|------|---------|---------|------|
| 刀兵 | 騎兵 | 槍兵 | バランス型近接 |
| 騎兵 | 槍兵 | 刀兵 | 高機動近接 |
| 槍兵 | 刀兵 | 騎兵 | 対刀特化近接 |
| 弓兵 | — | 刀・騎・槍（近接全般） | 遠距離攻撃。近接からは大ダメージを受ける |
| 衝車 | 城壁 | — | 攻城専用。都市攻城で大ダメージ |
| 投石 | — | — | 弓兵より長射程。攻撃力はやや低め |

### 兵装切り替え

- 兵種は3段階の兵装切り替えが可能（武将の欠片を消費）
- 2段階目：紫武将欠片、3段階目：赤武将欠片が必要
- 武将ごとに適正兵種が設定されており、適正兵種でダメージ・移動速度にバフが発生

## 4. スキルシステム

### スキルタイプ分類

| タイプ | 特徴 | 発動条件 |
|--------|------|---------|
| **怒気技（怒気スキル）** | 高倍率・範囲ダメージ。**主将のみ発動** | 怒気ゲージが満タン時 |
| **連鎖スキル** | 通常攻撃後に確率発動。発動率×倍率でDPS決定 | 通常攻撃後（確率） |
| **能動スキル** | ターンごとに確率発動の爆発力スキル | 各行動ごと（確率） |
| **パッシブスキル** | 常時効果。ステータス強化・条件付きバフなど | 常時 or 条件達成時 |

### 連鎖スキル計算式

```
期待DPS = 通常攻撃回数 × 連鎖発動率 × 連鎖倍率
```

### 軍師スキルの種類

- **全体連撃付与型**: 王彦章 — 全将領に75%確率で連撃状態2秒付与（刀・騎は15%減、弓は40%減）
- **連鎖強化型**: 樊噌 — 連鎖発動率+5%、連鎖ダメージ+20%（武力加算）
- **回復・バフ型**: 回復スキル・全体バフを付与する軍師
- **特定将領強化型**: 特定の武将や兵種を強化する軍師

### 武将3つ目のスキル解放

固有スキルLv7以上で解放可能

## 5. 推奨武将・スキル評価

### 連鎖武力部隊

| 役割 | 武将 | 評価理由 |
|------|------|---------|
| 主将（最有力） | 趙雲 | 多段連鎖型固有スキル。連鎖回数を増やすほど期待値が伸びる。龍胆との相性が極めて高い |
| 主将（代替） | 岳飛 | 35% / 250%の高倍率連鎖。怒気減少効果付きで安定と妨害を両立 |
| 副将 | 岳飛 | 趙雲と並ぶ2トップ枠。純火力＋怒気妨害 |
| 軍師（最優先） | 王彦章 | 全体に連撃付与。連鎖部隊の回転数を根本から底上げ |
| 軍師（代替） | 樊噌 | 連鎖発動率上昇。胆力無視による安定火力。純DPS寄せに強力 |

### 推奨サブスキル（連鎖部隊）

| スキル名 | 特徴 |
|---------|------|
| 龍胆 | 趙雲専用級。主将固定枠 |
| 閃撃 | 最高倍率枠。純火力要員 |
| 槍で三関を鎮める | 発動率減衰型。安定高火力枠 |
| 三軍に冠たる勇 | 敵主将アクティブ発動率低下。火力＋妨害の両立 |
| 肉弾戦 | 連撃付与・全ダメ15%増（王彦章採用時は優先度低下） |
| 戦で戦を養う | 長期戦向け。AOE対策 |
| 天門を大いに破る | 長期戦向け。AOE対策 |

| 5編成タイプ | 対応する warrior_roles.role |

| 連鎖武力 | 物理系パッシブ（副将）、弓（兵種適性） |

### skill_type 分布調査

```sql
SELECT skill_type, COUNT(*) as cnt FROM skills GROUP BY skill_type ORDER BY cnt DESC;
```

| skill_type | 件数 | squadScorerでの用途 |
|------------|------|-------------------|
| パッシブ | 164 | 常時強化スキル。連鎖・怒気・耐久問わず汎用 |
| 能動 | 76 | 能動武力 / 能動知力 部隊の判別に使用 |
| 連鎖 | 15 | 連鎖武力部隊の判別に直接使用可能 |
| 怒気技 | 6 | 怒気武力部隊の判別に直接使用可能 |

**判別可能性**: `skill_type` フィールドで連鎖（15件）と怒気技（6件）は直接判別可能。能動（76件）は武力・知力の区別のためさらに `warrior_roles.role` との組み合わせが必要。
