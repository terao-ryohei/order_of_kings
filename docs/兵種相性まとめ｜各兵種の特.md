基本兵種と相性システム
王の勅命では、兵種ごとに特徴と相性が設定されています。

戦闘では兵種相性を意識することで、戦闘を有利に進めることが可能です。

刀兵・騎兵・槍兵の3すくみ
刀兵・騎兵・槍兵は、3すくみの相性関係になっています。

それぞれの兵種に対してダメージ増加や軽減効果があり、編成時に重要な要素となります。

3すくみ関係


・刀兵 → 騎兵に強い
・騎兵 → 槍兵に強い
・槍兵 → 刀兵に強い

弓兵の特徴
弓兵は、遠距離から攻撃できることが最大の特徴です。

刀兵・騎兵・槍兵から攻撃を受けると大きなダメージを受けますが、安全な距離から攻撃できるメリットがあります。


衝車の特徴
衝車は戦闘用ではなく、攻城戦で活躍する兵種です。

都市攻城時に大きなダメージを与えることができ、高レベルの攻城戦では重要な役割を持ちます。

投石の特徴
投石は弓兵よりもさらに射程が長い兵種です。

遠距離から攻撃できますが、その分攻撃力はやや低めに設定されています。

武将と兵種適性システム
王の勅命では、武将ごとに適正兵種が設定されています。

適正兵種を使用することで、ダメージや移動速度にバフ効果が発生します。



武将適性の重要性
武将の適正兵種を活用することで、戦闘を有利に進めることが可能です。

特に戦闘では、兵種相性と武将適性の両方を意識することが重要です。

まとめ
王の勅命では、兵種相性と武将適性の組み合わせが重要になります。

状況に応じて最適な兵種と武将を選択することで、戦闘を有利に進めることができます。
